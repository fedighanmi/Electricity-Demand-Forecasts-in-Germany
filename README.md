# Energy Demand Modelling in Germany

This repository contains the work developed as part of the *Case Studies 2* module in the **Master in Data Science at Technical University of Dortmund**.
The goal of the project is to preprocess, analyze, model and derive conclusions from the energy demand data for Germany.

---

## üìÅ Project Structure

### **`external_data/`**

Contains all raw datasets used throughout the project. These files represent the original inputs for analysis and modeling.

### **`derived_data/`**

Includes all processed or transformed datasets generated during the workflow. These files were created through cleaning, feature engineering, or aggregation steps.

### **`utils.py`**

A collection of helper functions used in the Jupyter notebook.
These functions support data loading, preprocessing and evaluation a modular way.

### **`reports/`**

Contains the written project report, including:

* Introductions
* Data methodology
* Data Analysis
* Modelling of the Data
* Results Presentation
* Discussion of findings
* All relevant references
